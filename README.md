# Klotski-POS---Offline-First-PWA
Klotski POS - Offline-First PWA: Un MVP de Punto de Venta ágil y 100% frontend diseñado para comercios minoristas. Integra escáner de código de barras, cobros múltiples, gestión de cuentas corrientes y backups encriptados (AES), funcionando completamente sin conexión a internet.
